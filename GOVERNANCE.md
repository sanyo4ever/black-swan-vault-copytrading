# Governance

## Project model

Black Swan Vault is currently a founder-led open-source project with community contributions.

## Maintainers

- @sanyo4ever (project owner and release maintainer)

## Decision process

- Major architecture or policy changes are proposed via GitHub Issues/PRs
- Maintainer reviews technical fit, security impact, and operational cost
- Accepted proposals are merged into `main` and documented in `PROJECT_MEMORY.md`

## Release process

- `main` is the active branch
- Production deploys are performed after validation and smoke checks
- Critical fixes may be fast-tracked

## Community expectations

- Keep discussions technical and respectful
- Provide reproducible bug reports
- Prefer small, focused pull requests
